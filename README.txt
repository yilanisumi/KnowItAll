To deploy and run Know It All:

1) Download Xampp version 7.1.9 from https://www.apachefriends.org/download.html
2) Install Xampp
3) Unzip the file and put the 'www' folder inside ../xampp/htdocs
4) Launch Xampp and start Apache and MySQL in the Xampp control panel
5) Click Admin next to MySQL to go to the localhost/phpMyAdmin page
6) In phpMyAdmin create a new database and name it "knowitall_test"
7) Import the knowitall_test.sql file included in the 'www' zip file into knowitall_test database
8) If all the setup are done correctly, the website can now be accessed at "http://localhost/www/knowitall"
